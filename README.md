# 🌤️ App de Clima

Aplicativo web simples que mostra a temperatura e o clima em tempo real de qualquer cidade.

## 🚀 Funcionalidades

- Buscar cidade  
- Mostrar temperatura  
- Exibir condição do tempo  

## 🛠️ Tecnologias

- HTML  
- CSS  
- JavaScript  
- API de clima  

## 📦 Como usar

1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/seu-repositorio.git
```

2. Abra o arquivo `index.html` no navegador  

## 🔑 API

Adicione sua chave no JavaScript:
```javascript
const apiKey = "SUA_API_KEY_AQUI";
```
